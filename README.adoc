|===
| Op name | Required | Optional | Return

| iced-version
|
|
| `:version`

| lint-file
| `:file`,`:linters`
|
| `:lint-warnings`, `:error` (if occured)

| grimoire
| `:platform`, `:ns`, `:symbol`
|
| `:content`, `:http-status` (if errored)

| system-info
|
|
| `:user-dir`, `:file-separator`, `:project-name`

| project-namespaces
|
| `:prefix`
| `:namespaces`

| project-functions
|
| `:prefix`
| `:functions`

| format-code-with-indents,
| `:code`, `:indents`
|
| `:formatted`
